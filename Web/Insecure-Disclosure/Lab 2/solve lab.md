# Lab 2 – Information disclosure on debug page

## Lab Link
https://portswigger.net/web-security/information-disclosure/exploiting/lab-infoleak-on-debug-page

## Description

This lab shows how exposed debug pages can reveal sensitive server configuration and credentials.

## Solution

1. Explore the application and look for hidden endpoints.
2. A debug page is discovered that exposes internal configuration details.
3. Inspect the information displayed on the debug page.
4. Sensitive credentials such as the database password are revealed.

## Result

Submit the discovered password in the solution field to complete the lab.
