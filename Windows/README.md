# Windows & Privilege Escalation Labs

This section documents hands-on Windows exploitation and privilege escalation practice performed in lab environments.

## Topics Covered

- Local privilege escalation
- Service misconfiguration exploitation
- Weak permissions abuse
- Credential discovery techniques
- SYSTEM privilege acquisition scenarios

## Tools Used

- PowerShell
- WinPEAS
- Metasploit
- Windows command-line utilities

## Skills Demonstrated

- Post-exploitation enumeration
- Privilege escalation methodology
- Misconfiguration analysis
- Security impact assessment

All exercises were performed in isolated testing labs.
