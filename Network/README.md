# Network Security Labs

This folder contains practical network security and vulnerability assessment exercises performed in controlled lab environments.

## Scope

- Network enumeration and host discovery
- Service identification and port analysis
- Vulnerability scanning and validation
- Misconfiguration discovery
- Attack surface mapping

## Tools Used

- Nmap
- Nessus
- Wireshark
- Netcat
- Burp Suite (proxy analysis when required)

## Skills Demonstrated

- Network reconnaissance methodology
- Service enumeration workflows
- Risk identification and documentation
- Security assessment reporting

All activities were conducted in simulated environments for educational purposes.
