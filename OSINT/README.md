# Reconnaissance & OSINT Labs

This folder includes reconnaissance and open-source intelligence (OSINT) exercises used to simulate real penetration testing preparation phases.

## Activities

- Target information gathering
- Subdomain discovery
- Public exposure analysis
- Technology stack identification
- Attack surface mapping

## Tools Used

- Google Dorking
- Whois lookup
- DNS enumeration tools
- Online OSINT platforms

## Skills Demonstrated

- Structured reconnaissance workflow
- Intelligence analysis
- Target profiling
- Documentation of findings

All data collection was conducted on authorized lab targets.
